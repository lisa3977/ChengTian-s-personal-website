# 个人作品集网站

这是一个功能完整的个人作品集网站，具有可编辑的模块化设计。网站采用从上到下的布局，支持编辑模式、贴纸系统、鼠标粒子特效等高级功能。

## 功能特性

### 1. 视觉设计
- **阳光天蓝色渐变背景**：首页使用 `linear-gradient(135deg, #e0f7fa 0%, #b3e5fc 100%)`
- **动态几何装饰**：50+个随机几何图形，使用贝塞尔曲线动画
- **莫兰迪色盘**：12色活力色盘，统一视觉风格
- **Noto Sans SC字体**：中文字体支持，禁用装饰性字体

### 2. 编辑系统
- **三击触发**：在页面空白处快速三击进入编辑模式
- **密码保护**：默认密码为 "password"
- **安全问题找回**：忘记密码可通过安全问题重置
- **项目块排序**：拖拽重新排序内容区块
- **贴纸系统**：支持文本、图片、视频贴纸
- **文本编辑**：支持字号、颜色、超链接编辑

### 3. 特效系统
- **鼠标粒子特效**：Q弹跟随效果，点击交互
- **滚动动画**：区块淡入上浮动效
- **导航同步**：滚动时自动高亮对应导航项
- **响应式设计**：适配各种屏幕尺寸

## 使用方法

### 1. 浏览模式
- 打开 `index.html` 即可浏览网站
- 默认显示从上到下的布局：首页 → 关于我 → 项目作品 → 专业技能 → 联系我
- 导航栏支持平滑滚动到对应区块

### 2. 进入编辑模式
1. 在页面空白处（非按钮、输入框等可交互元素）**快速三击**
2. 输入密码：**password**
3. 点击确认进入编辑模式

### 3. 编辑功能
- **排序项目块**：拖动左侧面板中的项目标题重新排序
- **调整高度**：拖动项目块底部的调整手柄（仅支持向下延长）
- **删除项目块**：点击项目块右上角的 × 按钮
- **添加内容**：点击右下角 + 按钮，选择添加文本、图片、视频或新项目块
- **编辑文本**：双击文本进入编辑模式，使用上方工具条调整字号、颜色、添加链接
- **管理贴纸**：拖拽贴纸到任意位置，自动吸附到项目块

### 4. 退出编辑模式
- 点击右下角的 ✕ 按钮退出编辑模式
- 所有编辑内容自动保存到浏览器本地存储

## 技术实现

### 布局系统
- **从上到下布局**：使用 `display: block` 实现垂直流式布局
- **响应式设计**：使用媒体查询适配移动设备
- **CSS Grid/Flexbox**：用于内部组件布局

### 状态管理
- **本地存储**：使用 `localStorage` 保存编辑内容
- **编辑模式状态**：通过 `body.edit-mode` 类控制UI显示
- **密码加密**：安全问题答案使用加密存储

### 交互特效
- **贝塞尔曲线动画**：使用 `cubic-bezier()` 实现流畅动画
- **Intersection Observer**：检测元素进入视口触发动画
- **弹簧物理算法**：鼠标粒子跟随效果

## 部署说明

### GitHub Pages 部署
1. 将 `myonly` 文件夹推送到GitHub仓库
2. 在仓库 Settings → Pages 中启用GitHub Pages
3. 选择 `main` 分支和 `/myonly` 文件夹
4. 访问生成的URL即可查看网站

### 本地运行
1. 使用任何HTTP服务器（如Python、Node.js、Live Server）
2. 或直接双击 `index.html` 在浏览器中打开

## 注意事项

1. **编辑内容存储**：所有编辑内容仅保存在当前浏览器的 `localStorage` 中
2. **跨设备同步**：如需跨设备访问编辑内容，需要手动同步代码
3. **媒体文件**：上传的图片/视频文件保存在 `assets/images/` 和 `assets/videos/` 文件夹
4. **移动端适配**：移动设备自动禁用复杂粒子特效，使用触摸事件替代

## 默认密码和安全
